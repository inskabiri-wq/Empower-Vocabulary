## Environment notes (for next time)

- The Write-tool `/tmp` is a sandbox separate from the Windows node process
  and from git-bash `/tmp`. Playwright test scripts must be written to a
  **shared real path** (the project dir works) so `run.js` can `require` them.
- The actual skill runner is at `C:\Users\kabir\.agents\skills\playwright\run.js`
  (`.claude\skills\playwright` is a junction to it).
- Pass `run.js` an **absolute Windows/forward-slash path** it can `fs.existsSync`,
  not a `/tmp/...` path.
